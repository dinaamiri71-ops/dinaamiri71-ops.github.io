## Что такое Git?

Git — распределённая система управления версиями. Она позволяет отслеживать изменения в коде и работать в команде.

## Основные команды

- git init — создать репозиторий
- git add . — добавить файлы
- git commit -m "текст" — сохранить изменения
- git push — отправить на GitHub
